##### 四大函数式接口: 
1. function 函数式接口
2. predicate 断定型接口
3. consumer 消费型接口
4. supplier 供给型接口

### 函数式接口:
```java
/**
 * 函数式接口:接收一个入参，然后返回一个结果
 * <p>
 * 只要是函数式接口，都可以用lambda表达式简化
 */
public class FunctionalDemo {
    public static void main(String[] args) {
        Function<String, String> function = new Function<String, String>() {
            @Override
            public String apply(String str) {
                return str.toUpperCase();
            }
        };

        Function<String, String> function1 = (str) -> {
            return str.toUpperCase();
        };

        System.out.println(function.apply("abc"));
        System.out.println(function1.apply("def"));
    }
}
```
运行结果:
```
ABC
DEF
```

### 断定型接口:
```java
/**
 * 断定型接口:接收一个入参，根据是否符合规则返回 true 或 false
 */
public class PredicateDemo {
    public static void main(String[] args) {
        Predicate predicate = new Predicate() {
            @Override
            public boolean test(Object o) {
                return false;
            }
        };
        Predicate predicate1 = (x) -> {
            return false;
        };
        System.out.println(predicate.test(new Object()));
        System.out.println(predicate1.test(new Object()));
    }
}
```
运行结果:
```
false
false
```
